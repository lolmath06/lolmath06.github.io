# Notes Techniques

### Caméras : 
- Les scripts utilisent cv2.VideoCapture pour accéder à la caméra. Assurez-vous que l'index de la caméra est correct.
### Modèles YOLO :
- Les fichiers YOLO (yolov8n.pt, yolov8n-pose.pt) doivent être téléchargés et placés dans le répertoire approprié.
### Images d'encodage :
- Les images utilisées pour l'encodage des visages doivent être placées dans le dossier img/.
